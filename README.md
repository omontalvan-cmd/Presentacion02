# Presentacion02
# Proyecto Módulo 2 – Trabajo Final

## Descripción
Aplicación desarrollada en **Streamlit** que integra conceptos fundamentales de programación:  
variables, condicionales, estructuras de datos, funciones, programación funcional y POO.  
Permite realizar un **Análisis Exploratorio de Datos (EDA)** sobre un dataset de una compañía de seguros, mostrando estadísticas descriptivas, visualizaciones y hallazgos clave.

## Estructura
- app.py
- requirements.txt
- README.md
- dmc.png
- InsuranceCompany.csv

## Instrucciones de ejecución
1. Instalar dependencias:
   pip install - requirements.txt
2. Ejecutar la aplicación:
   streamlit run app.py
